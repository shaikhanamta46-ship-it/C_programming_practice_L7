#  C Arrays & Pointers in C

A collection of beginner-friendly C programs covering arrays, pointers, pointer arithmetic, and 2D arrays.

## Topics Covered

- Arrays in C
- Array Input & Output
- Array Initialization
- Pointer Arithmetic
- Arrays and Memory Addresses
- Arrays Using Pointers
- Two-Dimensional Arrays (2D Arrays)

## Programs Included

### 1. Basic Arrays
Store and access values using array indexing.

### 2. Array Input & Output
Take user input and display array elements using loops.

### 3. Array Initialization
Initialize arrays during declaration.

### 4. Pointer Arithmetic
Understand how pointers move through memory.

### 5. Arrays in Memory
View the memory addresses of array elements.

### 6. Arrays Using Pointers
Access array elements using pointers.

### 7. 2D Arrays
Create and display a matrix using nested loops.

## Learning Outcomes

- Understand array declaration and initialization
- Take input and display array elements
- Learn how arrays are stored in memory
- Understand the relationship between arrays and pointers
- Practice pointer arithmetic
- Work with two-dimensional arrays
